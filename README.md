Quality - готовый каркас дипломного проекта

Что внутри:
- WinForms проект на C# (.NET Framework 4.8)
- формы с файлами .cs + .Designer.cs
- SQL-скрипты init.sql и seed.sql
- роли: Администратор, Врач, Менеджер по качеству
- авторизация и переход по роли
- заготовка для работы с MySQL через MySql.Data

Тестовые учётные записи:
- admin / admin123
- doctor1 / doctor123
- doctor2 / doctor123
- quality / quality123

Что нужно сделать в Visual Studio:
1. Открыть Quality.sln
2. Восстановить NuGet пакеты
3. Убедиться, что MySQL доступен
4. При необходимости поправить App.config
5. Запустить проект
